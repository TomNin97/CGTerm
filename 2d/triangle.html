This is a triangle!

WebGL creates shapes by drawing points, lines and triangles through code. Triangles are one of the most basic shapes and are foundations for many more. Triangles are created from 3 vertices. The Buffer Data copies these points and sends them to the Position Buffer. This initlializes the code for the triangle. Finally, the program draws the triangle with the given vertices and color.
