This is a square!

WebGL creates shapes by drawing points, lines and triangles through code. A square is created by combining two 90 degree triangles. The WebGL coordinate system ranges from -1,-1 to 1,1 with 0,0 being in the center. To create a square, you will need to create an array, or in WebGl, a buffer. Then fill the array with 12 coordinates. 6 points in 2D will form two triangles. The second triangle contains the same coordinates as the first triangle, but negative so that it will be a direct invert.
