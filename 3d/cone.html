This is a cone!

Since webGl creates shapes primarily with vertices, curved shapes are created using many squares and triangles.

As such, the cone buffer class takes in several more parameters than a simple cube.

The parameters in order are: The radius of the bottom circle, the radius of the top circle (for a cone, this value should be zero), the height of the cone, the number of subdivisons or squares horizontally around the cone, the number of subdivison squares vertically on the cone, and finally whether or not there is a top circle, and another bool to determine a bottom circle.

By setting the Top Circle Bool parameter to true, and providing a value other than zero for the radius of the top circle, a cylinder can be created in a similar manner. You can use the controls below the canvas to demonstrate this.
