This is an Example of a Cube

WebGL creates shapes primarily with vertices

A cube is then created by creating a shape made of multiple triangles. These triangles, with 45, 45, and 90 degree angles, are created with one another to make squares. A total of 8 squares are made with parrallel and/or perpendicular edges to each other to create a cube.

Since all the cube requires is squares made of triangles, the cube buffer class only takes in two parameters, the webgl canvas and the size of the squares.

This can be changed by setting one of the values(the cube size) to a variable as shown below. You can change the size of the cube by selecting the size of each square within the cube
